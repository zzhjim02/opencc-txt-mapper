=======================================
     TXT文件繁简转换器 - 使用说明
=======================================

【程序简介】
本程序是一个功能强大的TXT文件繁简转换工具，支持批量转换，
能够智能识别一对多和多对一的字符转换，自动检测文件编码，
并将所有输出文件统一转换为UTF-8编码。

【主要功能】
✓ 批量转换多个TXT文件
✓ 支持繁体转简体和简体转繁体
✓ 自动检测文件编码（UTF-8、GBK、GB2312、BIG5等）
✓ 统一输出UTF-8编码
✓ 智能识别一对多和多对一转换
✓ 详细的转换统计和示例
✓ 显示编码检测置信度
✓ 自定义输出目录和文件命名
✓ 实时进度显示
✓ 无需安装Python环境，直接运行

【使用方法】
1. 双击运行"繁简转换器.exe"
2. 点击"选择TXT文件"按钮，选择要转换的文件（支持多选）
3. 选择转换方向：
   - 繁体转简体：将繁体中文转换为简体中文
   - 简体转繁体：将简体中文转换为繁体中文
4. 点击"浏览"按钮选择输出目录
5. 设置文件命名方式：
   - 添加后缀：在原文件名后添加指定后缀（默认为_converted）
   - 覆盖原文件：直接覆盖原文件（请谨慎使用）
6. 点击"开始转换"按钮开始批量转换
7. 在转换日志区域查看详细的转换结果和统计信息

【编码识别功能】
程序会自动检测以下编码：
- UTF-8（推荐）
- GBK/GB2312（简体中文）
- BIG5（繁体中文）
- UTF-16/UTF-16-LE/UTF-16-BE
- ASCII
- 其他常见编码

检测结果显示：
- 原始文件编码类型
- 编码检测置信度
- 输出文件统一为UTF-8编码

【转换统计信息】
程序会显示以下统计信息：
- 原始文件编码和置信度
- 输出文件编码（统一UTF-8）
- 总字符数和变化字符数
- 一对多转换数量（一个繁体字对应多个简体字）
- 多对一转换数量（多个简体字对应一个繁体字）
- 具体转换示例
- 每个文件的转换详情

【支持的转换示例】
一对多转换（繁体→简体）：
- 乾浄 → 干浄
- 髮型 → 发型
- 麵包 → 面包
- 後來 → 后来
- 併購 → 并购

多对一转换（简体→繁体）：
- 干净 → 乾淨
- 发型 → 髮型
- 面包 → 麪包
- 后来 → 後來
- 并购 → 併購

【编码转换示例】
原始文件：GBK编码
程序检测：编码类型: GBK (置信度: 100%)
转换后文件：UTF-8编码
输出格式：统一的UTF-8编码，确保兼容性

【注意事项】
✓ 程序会自动检测文件编码，无需手动指定
✓ 所有输出文件统一使用UTF-8编码
✓ 程序支持多种常见中文编码
✓ 如果编码检测失败，程序会尝试其他编码
✓ 覆盖原文件模式会永久修改原文件，请谨慎使用
✓ 转换过程中请不要关闭程序
✓ 如遇到问题，请检查文件是否可读可写

【系统要求】
- 操作系统：Windows 7/8/10/11
- 内存：建议2GB以上
- 磁盘空间：至少50MB可用空间

【技术支持】
如遇到问题或需要帮助，请检查：
1. 文件是否为常见的中文编码
2. 文件是否可读可写
3. 磁盘空间是否充足
4. 输出目录是否有写入权限
5. 查看转换日志中的编码检测信息
