# TXT文件繁简转换器

一个带图形用户界面的批量TXT文件繁简转换工具。

## 功能特点

- 批量选择多个TXT文件进行转换
- 支持繁体转简体和简体转繁体
- 自定义输出目录
- 灵活的文件命名方式（添加后缀或覆盖原文件）
- 实时显示转换进度
- 详细的转换日志
- 线程处理，界面不会卡顿

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python converter.py
```

## 使用说明

1. **选择文件**：点击"选择TXT文件"按钮，选择一个或多个要转换的TXT文件
2. **设置转换方向**：
   - 繁体转简体：将繁体中文转换为简体中文
   - 简体转繁体：将简体中文转换为繁体中文
3. **选择输出目录**：点击"浏览"按钮选择转换后文件的保存位置
4. **设置文件命名**：
   - 添加后缀：在原文件名后添加指定后缀（默认为_converted）
   - 覆盖原文件：直接覆盖原文件（注意：此操作不可逆）
5. **开始转换**：点击"开始转换"按钮开始批量转换
6. **查看日志**：在转换日志区域可以查看详细的转换结果

## 技术说明

- 使用OpenCC库进行繁简转换
- 使用Tkinter构建图形用户界面
- 使用多线程处理转换，避免界面卡顿
- 支持UTF-8编码的TXT文件

## 注意事项

- 请确保所有TXT文件使用UTF-8编码
- 覆盖原文件模式会永久修改原文件，请谨慎使用
- 转换过程中请不要关闭程序
